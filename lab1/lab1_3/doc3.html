Атрибут target

Атрибут target определяет, где будет открыта ссылка.

_self - Открывает документ в текущем окне / вкладке (значение по умолчанию)

_blank - Открывает документ в новом окне или вкладке

_parent - Открывает документ в родительском фрейме

_top - Открывает документ во всем окне

Пример:



 
  
  Атрибут target
